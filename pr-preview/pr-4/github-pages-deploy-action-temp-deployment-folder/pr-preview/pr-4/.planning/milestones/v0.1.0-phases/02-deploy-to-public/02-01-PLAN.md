2. Create `assets/og-image.svg`: A 1200x630 SVG image for social sharing. Dark background (#0a0a0a), include the crab mascot (embed or reference key paths from crab-mascot.svg), the text "AgentLinux" in large type, and "Linux, for agents" as subtitle. Use the same color palette as the site (white/light text on dark). This is an SVG that can be referenced as the OG image — GitHub Pages will serve it. Note: Some social platforms may not render SVG for og:image; if this is a concern, the user can convert to PNG later, but SVG will work for direct links and many modern platforms.

```
User-agent: *
Allow: /

```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://agentlinux.org/</loc>
    <lastmod>2026-03-09</lastmod>
    <priority>1.0</priority>
  </url>
</urlset>
```
