# Phase 36 Plan — github-mcp (remote-http + cross-agent MCP)

**Requirement:** MCP-03 · **Enabler folded in:** ENABLE-02 remote-http + shared
cross-agent MCP registration helper.
**Depends on:** Phase 35 (ENABLE-02 mcp source_kind + secret convention).

## Goal (goal-backward)

`agentlinux install github-mcp` registers GitHub's hosted remote MCP server
(`https://api.githubcopilot.com/mcp/`) into **every installed MCP-capable agent**
(claude-code, codex, gemini-cli, opencode, qwen-code) using an env-var
**reference** for the PAT (never the literal token); `remove` deregisters from
all of them with no residue; the mandatory `GITHUB_MCP_PAT` is supplied
post-install; ≥1 bats @test proves the lifecycle + no-baked-token + no-Docker
shape — Docker green.

## Task breakdown

### 36-01 — schema + types: remote-http mcp support
- `plugin/catalog/schema.json`: add optional `endpoint_url` (string,
  `format: uri`, `pattern: ^https://`) to `$defs/agent`. `pinned_version` stays
  required (semver) — for a remote entry it names the curated upstream
  `github-mcp-server` release the hosted endpoint is validated against (ADR-011
  curated combo), NOT a package to fetch.
- `plugin/cli/src/types.ts`: add `endpoint_url?: string`.
- `plugin/cli/test/schema.test.ts` (+ fixture): a remote mcp entry with
  `endpoint_url` + `requires_secret:true` validates; `endpoint_url` must be
  https.

### 36-02 — shared cross-agent helper `plugin/catalog/lib/mcp-register.sh`
Per-agent writers, each gated on agent presence (`command -v <bin>`), never-bake
(env-var reference only), idempotent, atomic writes. API:
- `al_mcp_register_http <server> <url> <secret_env>` — fan out to all present agents.
- `al_mcp_deregister <server>` — remove from all present agents.
- `al_mcp_targets` — echo the present agents (for logging).
Per-agent (reference = `Bearer ${<secret_env>}`, literal, unexpanded):
- **claude-code** (`claude`): remove-then-add `claude mcp add --transport http
  <server> <url> --scope user --header "Authorization: Bearer \${<env>}"`.
- **codex** (`codex`): marker-delimited block in `~/.codex/config.toml`:
  `[mcp_servers.<server>]` `url=…` `bearer_token_env_var="<env>"` (token off-disk).
- **gemini-cli** (`gemini`): jq merge `~/.gemini/settings.json`
  `.mcpServers[<server>]={httpUrl,headers:{Authorization:"Bearer ${<env>}"}}`.
- **opencode** (`opencode`): jq merge `~/.config/opencode/opencode.json`
  `.mcp[<server>]={type:"remote",url,enabled:true,headers:{…}}`.
- **qwen-code** (`qwen`): jq merge `~/.qwen/settings.json` (gemini schema).
- **Never-bake guard:** after fan-out, scan each written config for a literal
  token (`ghp_`, `github_pat_`, 40-hex) → fail loud if found.
- RETURN-trap functrace guard mirrors prebuilt-binary.sh. shellcheck clean.

### 36-03 — github-mcp recipe pair + catalog entry
- `plugin/catalog/agents/github-mcp/install.sh`: source the helper (via
  `AGENTLINUX_CATALOG_DIR/lib/mcp-register.sh`), `al_mcp_register_http github-mcp
  <url> GITHUB_MCP_PAT`, print the mandatory PAT instruction, assert ≥1 agent
  targeted (else clear pointer: install a coding agent first).
- `uninstall.sh`: `al_mcp_deregister github-mcp`; assert no residue in any config.
- `catalog.json`: `source_kind:mcp`, `pinned_version:"1.5.0"`,
  `endpoint_url:"https://api.githubcopilot.com/mcp/"`, `requires_secret:true`,
  `secret_env:"GITHUB_MCP_PAT"`, Apache-2.0, `post_install_verify` (claude branch).

### 36-04 — bats gate + docs + planning

  → symmetric remove → no residue → idempotent re-remove → absent-agent skip.
- `docs/internals/catalog.md`: remote-http + cross-agent MCP section.
- `.planning`: REQUIREMENTS MCP-03 → Done; ROADMAP 36 Complete; STATE narrative;
