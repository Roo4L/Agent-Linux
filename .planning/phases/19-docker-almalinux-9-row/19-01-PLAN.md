        bats sudo \
        dbus-broker \

  `dnf install curl` triggers the curl/curl-minimal conflict, RESEARCH Pitfall 1);
  add `procps-ng` (optional `ps` for debug). All names live-verified on
  almalinux:9 2026-06-28.

  ` || true` to the trailing `.timer` line (resolved/networkd aren't installed

- ssh host keys: `RUN mkdir -p /run/sshd && ssh-keygen -A` verbatim (line 118).
- DROP the `locale-gen`/`update-locale` RUN (Ubuntu lines 120-122) ENTIRELY —
