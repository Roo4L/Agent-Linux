| installer (root) → install user (agent) | Detection probes that need to resolve as the install user (`npm config`, `command -v <agent>`, `test -w <home>`) cross from root context into agent context via `as_user`. Output of those calls is UNTRUSTED and must never be eval'd or unquoted-expanded. |
