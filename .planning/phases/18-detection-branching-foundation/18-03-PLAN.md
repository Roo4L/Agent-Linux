|----------|-------------|
| network → 30-nodejs NodeSource setup | curl-pipe-bash of setup_22.x runs as root; installs a repo + GPG key |
| installer → /etc/sudoers.d/agentlinux | the drop-in grants `agent ALL=(ALL) NOPASSWD: ALL` — a privilege-escalation surface |
| installer → /etc/locale.conf | a root-written system config consulted by every login shell |

No EL9 `Defaults` is added by this installer |
| T-18-10 | Tampering | NodeSource RPM repo GPG trust | mitigate | setup_22.x writes `gpgkey=…ns-operations-public.key` + `gpgcheck=1` (verified verbatim 2026-06-28); dnf verifies every package signature. HTTPS+`curl -fsSL` guards the fetch. Same posture as the validated deb path (ADR-005) |
| T-18-11 | Tampering | AppStream module reset masking a hostile pre-existing stream | accept | `dnf -y module reset nodejs` only clears module *enablement*; the subsequent install is GPG-verified from NodeSource. RT-01 node>=22 hard-check is the post-condition guard |
